创建NuxtJs项目

准备环境

  • 拉取Docker镜像
docker pull node:20.3.0-bullsey

如果速度较慢，可设置先设置镜像源，然后拉取。

  • 设置镜像源 
{
  "registry-mirrors": [
    "https://registry.docker-cn.com",
    "https://cr.console.aliyun.com"
  ]
}
  • 创建容器
docker run -dt --privileged --gpus all ^
-v <path-to-nodejs-projects>:/root/nodejs-projects/ ^
--name nodejs-dev ^
node:20.3.0-bullseye

新建和运行项目

  • 容器中，新建nuxt项目，安装依赖的包
cd /root/nodejs-projects/
npx nuxi init <project-name>
cd <project-name>
npm install
  • 运行项目，用于开发调试。
npm run dev -- -o

部署项目

  • 生成静态网页，用于部署。
npx nuxi generate

将.output/public作为一个本地仓库push到github仓库或其他静态网站托管平台。


  • github仓库开启page服务。
